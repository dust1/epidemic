# epidemic 一个去中心化的存储节点
用于集群中存储文件。

#### 2021/04/27 更新：
第一版的功能如下：
1. 只支持文件的上传与下载
2. 文件的删除与垃圾回收放在2.0版本，但是当前需要实现逻辑删除的标记
3. 当存储节点满时将文件托付给其他节点这个功能也放在2.0实现

    2. 路由节点管理【未实现】
